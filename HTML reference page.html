HTML reference page
  • HTML: the syntax command for html is <html>
  • Header: the syntax command for a header is <head>
  • Title: the syntax command for a title is <title>
  • Body: the syntax command for a body is <body>
  • Headings: the syntax command for a heading is <h1> <h2>
  • Paragraph: the syntax command for a paragraph is <p>
  • Breaks: the syntax command for a break is <br>
  • Lines: the syntax command for a line is <hr>
  • Comments: the syntax command for a comment is <!-- -->
  • Links: the syntax command for a link is <a>
  • Images: the syntax command for an image is <img>
  • Lists: the syntax command for a list is <ul> <ol> <li>
  • Tables: the syntax command for a table is <table> <tr> <th> <td>
  • Blocks: the syntax command for a block is <div>
